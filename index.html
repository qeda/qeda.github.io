What is QEDA?

QEDA is a set of tools aimed to make electronic devices development more easy and funny.

They are completely free and opensource.

Getting started

Structure

QEDA consists of:

  • QEDA Core — an utility which provides a simple but highly customizable instrument for creating electronic component libraries for using in EDA software. May be used both as command line tool and as Node.js module.

    This is the engine of QEDA.

  • QEDA Library — open, free and constantly growing repository of ready-for-use electronic components.

    This is the fuel of QEDA.

Features

Let us point out some features that may attract you:

  • Simple electronic component definition

    Even if a component is not in library yet its definition reduces to creating short and human readable text file (we use YAML format for that purpose).

    Read more

  • Flexible customization

    We tried to take into account all settings you would like to deal with and grouped them to easily accessible parameters.

    Read more

  • Compliance with your corporate style

    All schematic symbols and land patterns will share common style according to your specific tunings. When your style changes the only thing you are to do is to regenerate libraries with new settings. You may adapt the look of schematic symbols to actual standard of your company or country.

    Read more

  • Accordance to leading industry standards (IPC, JEDEC, EIA, ISO, JEITA, GOST)

    We use IPC-7351 guidelines (even last IPC-7351C) to create land patterns. Housing specifications are partially borrowed from respective JEDEC and JEITA standards.
